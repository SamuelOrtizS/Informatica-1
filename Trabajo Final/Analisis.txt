---------------------------------------------------------------------
Proceso
Descripcion del proceso realizado
Se crean las constantes y matrices para representar las dos salas de cine.
Se identifica el tipo de silla segun la fila (general o preferencial).
Se calcula el precio de cada silla segun su tipo.
Se lee y valida la informacion ingresada por el usuario.
Se permite mostrar el estado de una sala con sus sillas libres o reservadas.
Se permite reservar una o varias sillas en la sala activa.
Se permite cancelar una reserva existente.
Se calcula el total recaudado por sala y el total general del sistema.
Se muestra un menu interactivo y se ejecuta la Opcion seleccionada hasta que el usuario elige salir.
